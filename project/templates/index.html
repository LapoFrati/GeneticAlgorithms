Welcome!

 

This website was built to create a user-friendly interface for a genetic algorithm that simulates the evolution of mutation rates. This simulation can be a base for answering questions on the evolution of evolvability for scientists, and it provides an interactive site for non-scientist interested in the topic.

 

In its current state, you can use this site to:

  • Learn more about mutations, mutation rates and the reasons why this topic is so controversial
  • Find the optimal mutation rate of the population of virtual agents and observe the outcome
  • Evolve the mutation rate itself to see whether the population can find the optimal one
  • Explore the topic with us! Look at the questions we are currently trying to answer.
